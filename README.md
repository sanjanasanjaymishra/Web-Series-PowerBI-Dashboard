# 🎬 Web Series Data Analysis - Power BI Dashboard

## 📌 Project Overview
This project analyzes web series data using Power BI to uncover insights about ratings, genres, and trends.

## 📊 Key Insights
- Genre-wise distribution of web series
- Comparison of ratings across different shows
- Most popular genres
- Year-wise release trends
- Top-rated web series

## 📈 Dashboard Features
- Interactive visuals
- Clean and professional layout
- Dynamic filtering options
- KPI indicators

## 🛠 Tools Used
- Microsoft Power BI
- Data Cleaning
- Data Visualization

## 📂 Files Included
- Power BI Dashboard (.pbix file)
- Dashboard Screenshot

## 🎯 Objective
To analyze entertainment data and create an interactive dashboard for better understanding of web series trends.
